### How to push your code to this repository

[Video Explanation of the setup](https://drive.google.com/file/d/1pGKJvLxD3V5zXuTILn0Tv0Nsdbl5pTG5/view?usp=sharing)

1. Clone the repo 

```
git clone <repo-name>
```
2. Create a branch 

```
git branch -b <branch-name>
```
3. Code your solution
4. Push your changes
